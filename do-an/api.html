API Documentation

Welcome to the API Documentation. Use the following endpoints to interact with our services.

1. Create a New Order

Endpoint: POST /api/order

Request Body: { "service": "like", "quantity": 1000, "price": 20 }

Response: { "status": "success", "message": "Order created successfully" }

Status Code: 200 OK

2. Get Order Status

Endpoint: GET /api/order/{order_id}

Request Example: /api/order/12345

Response: { "status": "processing", "message": "Order is being processed" }

Status Code: 200 OK

3. Cancel Order

Endpoint: DELETE /api/order/{order_id}

Request Example: /api/order/12345

Response: { "status": "success", "message": "Order canceled successfully" }

Status Code: 200 OK

4. List Available Services

Endpoint: GET /api/services

Response: { "services": ["like", "comment", "follow", "share"] }

Status Code: 200 OK
